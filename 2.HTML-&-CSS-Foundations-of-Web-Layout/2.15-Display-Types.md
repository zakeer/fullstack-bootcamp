## ğŸš« 4. `display: none`

```css

  display: none;
}
```

### Behavior:

* The element **does not appear at all**
* **Not visible and not in document flow**
* Different from `visibility: hidden` (which hides it but still takes space)

### Use cases:

* Conditional UI rendering
* Dropdowns, modals, tabs

```html

```
